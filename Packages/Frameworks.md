## Computer Vision Frameworks
- LightlyTrain : https://github.com/lightly-ai/lightly-train

LightlyTrain은 레이블링되지 않은 데이터(unlabeled data)를 활용하여 컴퓨터 비전 모델을 효율적으로 사전 학습(pre-training)할 수 있게 해주는 오픈 소스 파이토치(PyTorch) 프레임워크입니다.

- scikit-image: Image processing in Python : https://github.com/scikit-image/scikit-image

scikit-image는 Python 프로그래밍 언어를 위한 오픈 소스 이미지 처리 라이브러리입니다. skimage라는 약칭으로도 불리며, 이미지 분석, 필터링, 변환 등 광범위한 기능을 제공합니다.

- scikit-opt : https://github.com/guofei9987/scikit-opt

scikit-opt (또는 sko)는 군집 지능(Swarm Intelligence) 및 기타 휴리스틱 알고리즘을 위한 강력한 오픈 소스 Python 라이브러리입니다. 최적화 문제, 특히 전역 최적화 문제를 해결하는 데 중점을 둡니다.  
유전 알고리즘(Genetic Algorithm), 입자 군집 최적화(Particle Swarm Optimization), 시뮬레이티드 어닐링(Simulated Annealing), 개미 군집 알고리즘(Ant Colony Algorithm), 인공 어군 알고리즘(Artificial Fish Swarm Algorithm) 등 다양한 휴리스틱 알고리즘을 구현합니다.
